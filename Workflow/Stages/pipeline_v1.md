# STUDIO Workflow Pipeline v1

This document defines the baseline 5-stage workflow process:

1. **Concept**
   - Define the core idea, target experience, and success criteria.
2. **Design**
   - Refine gameplay loops, systems, and user-facing behavior.
3. **Architecture**
   - Specify technical structure, module boundaries, and integration strategy.
4. **Implementation**
   - Build and integrate the defined systems incrementally.
5. **Testing**
   - Validate functionality, stability, and alignment with design goals.

## Flow

`Concept -> Design -> Architecture -> Implementation -> Testing`
